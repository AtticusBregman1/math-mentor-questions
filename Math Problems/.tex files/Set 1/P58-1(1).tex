\noindent In the figure below, \emph{L//M}. Find the measure of \angle \emph{a}.\footnote{Akita Prefecture}\\

\begin{center}
\section*{Solution}
\end{center}

\noindent \begin{math}
Answer: 65^\circ
\end{math}\\

\noindent Proof: Since corresponding angles are congruent, we can set up the equation: \(\angle a+40^\circ+75^\circ=180^\circ\). Solving for \emph{a}, we get \(\mathbf{\angle a=65^\circ}\).
